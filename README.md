# Twit Listener

## Install

If Homebrew isn't installed
```
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

If Vapor isn't installed
```
brew install vapor/tap/vapor
```

Install dependencies

```
vapor update
```

## Build & Run

```
vapor build
vapor run
```
